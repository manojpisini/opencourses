## About

This free, open-source course runs entirely on GitHub. No platform account
needed — just a GitHub account and a terminal.

**Tags:** `git` `version-control` `cli` `collaboration` `open-source`
**Duration:** 12-18 hours · **Maintainer:** @manoj

## 📋 Curriculum

| # | Stage | Duration | XP | Level |
|---|-------|----------|----|-------|
| 01 | 📝 [Git Foundations](curriculum/stage-01/README.md) | 2-3 hours | 100 XP | beginner |
| 02 | 🐛 Debugging with Git | 2-3 hours | 150 XP | beginner |
| 03 | 👁️ Code Review Workflow | 3-4 hours | 200 XP | intermediate |
| 04 | ⚡ Advanced Git Internals | 3-5 hours | 250 XP | intermediate |

## 🚀 How to Enroll

1. [Open an enrollment issue](../../issues/new?template=enroll.yml) with `course: git-mastery`
2. Get a welcome comment with your personalized learning path
3. Fork this repo and work on Stage 01
4. Submit via PR — automated grading runs instantly

## 📜 Certificate

Pass all 5 stages to earn a **cryptographically signed** completion certificate,
compatible with [Open Badges 3.0](https://www.imsglobal.org/spec/ob/v3p0/).
