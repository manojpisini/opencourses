# 📼 Git Mastery — Learning Playlist

## Stage 01: Git Foundations

**Type:** assignment · **Duration:** 2-3 hours · **XP:** 100

Set up your Git environment and learn the core workflow: init, add, commit,
log, diff, and status. Build a small project with at least 5 meaningful
commits, a feature branch, and a clean `.gitignore`.

**Resources:**

- [Git - The Simple Guide](https://rogerdudler.github.io/git-guide/)
- [Pro Git Book (free)](https://git-scm.com/book/en/v2)
- [Learn Git Branching (interactive)](https://learngitbranching.js.org/)
- [Git in 100 Seconds](https://www.youtube.com/watch?v=hwP7WQkmECE)

## Stage 02: Debugging with Git

**Type:** debugging · **Duration:** 2-3 hours · **XP:** 150

Track down and fix 3 seeded bugs using `git bisect`, `git log -S`, and `git
blame`. Document your debugging process in a `DEBUG.md` log.

## Stage 03: Code Review Workflow

**Type:** code-review · **Duration:** 3-4 hours · **XP:** 200

Write a comprehensive code review for a provided codebase. Produce a
structured `REVIEW.md` covering correctness, style, performance, and security.

## Stage 04: Advanced Git Internals

**Type:** assignment · **Duration:** 3-5 hours · **XP:** 250

Explore Git's object model (blobs, trees, commits, tags), write custom hooks,
use interactive rebase to clean up history, and configure a Git aliases
workflow.

## Stage 05: Final Project — Open Source Contribution

**Type:** project · **Duration:** 4-6 hours · **XP:** 400

Make a real contribution to an open-source project. Fork a repo, fix a
real issue, write tests, and submit a PR that follows the project's
contributing guidelines.
