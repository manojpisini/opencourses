Rust is a systems programming language that runs blazingly fast, prevents segfaults, and guarantees thread safety.

This course takes you from zero to writing production-quality Rust — covering the ownership model, the borrow checker, traits, generics, async/await, and the Tokio runtime.

## What You'll Build

By the end of this course you will have built:

- A command-line JSON processor (uses iterators, Result, Option)
- A concurrent file downloader (async/await, Tokio, channels)
- A simple HTTP server using Axum

## Prerequisites

- Comfortable in a terminal
- Some programming experience in any language
